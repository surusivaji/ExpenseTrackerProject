1. this project is developed by using servlets, jsp, hibernate, css, mysql.
2. in this project iam segregated the project into three layers that is dao layer, controller layer, model layers. dao layer contains the database logic here we can perform database operations in this we can perform crud operations on the database, controller layer contains the servlet classes here we can handle http request and responses through the forms, model layer contains the entity classes by using these entity classes we can create tables and also we can perform database operations on the database.
3. iam crated registration page based on this user can give details. those details saved on the database.
4. when ever user can login to the application we can performs operations like add expenses, delete expenses, edit expenses, remove expenses, edit user details, display user details, update user details.
5. if user forgot his password i can add feature that forgot password option
6. for success and failure messages i will use session object to print information
7. for creating user interfaces i will use jsp. because by using jsp i can write both java code and html code
8. to connect database with java application iam using hibernate framework
